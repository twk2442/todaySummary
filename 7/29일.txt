 react 엘리먼트
 

 엘리먼트는 컴토넌트의 구성요소

 React DOM은 해당 엘리먼트와 그 자식 엘리먼트를 이전의 
 엘리먼트와 비교하고 DOM을 원하는 상태로 만드는데 필요한 경우에만 
 DOM을 업데이트합니다.

 //

 컴포넌트:
 props(함수로치면 매개변수)를 입력받고 화면에 어떻게 표시되는지를
 기술하는 react 엘리먼트를 반환한다.

 !! 컴포넌트의 이름은 항상 대문자로 시작( 소문자로 시작시 DOM 태그로 인식)

모든 React 컴포넌트는 
자신의 props를 다룰 때 반드시 순수 함수처럼 동작해야 합니다.
